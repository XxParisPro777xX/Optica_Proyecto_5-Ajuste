# Proyecto 5: Simulador avanzado y ajuste de perfiles

Objetivo: construir una herramienta para ajustar perfiles de difracción variando forma, tamaño, orientación e impacto.

Tareas mínimas:
- Generar o usar una curva sintética.
- Definir modelos geométricos simples.
- Implementar chi-cuadrada o métrica equivalente.
- Explorar degeneraciones entre forma, tamaño, orientación e impacto.
